# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2024-01-01

### Added
- Initial release
- Check quota functionality with Allow2 API
- Enable/disable PlayStation Network access
- Play time reporting to Allow2
- Quota exceeded event trigger
- Quota renewed event trigger
- Session started event trigger
- Session ended event trigger
- Support for multiple child profiles
- PS4 and PS5 device support
- Network and configuration permissions
- Integration with PSN parental controls
